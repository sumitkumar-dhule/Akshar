## ✨ Features

*   **Interactive Flashcards:** Engaging, touchable flashcards for alphabets and numbers.
*   **Audio Pronunciation:** Tap on a card to hear the sound, helping with auditory learning.
*   **Multilingual Support:** Learn in three different languages:
    *   🇬🇧 English
    *   🇮🇳 Hindi (हिन्दी)
    *   🇮🇳 Marathi (मराठी)
*   **Kid-Friendly UI:** A clean, colorful, and intuitive interface designed specifically for young children with no distractions.
*   **Built with Modern Tech:** A 100% Kotlin codebase using the latest Android development tools.

## 🛠️ Tech Stack & Architecture

This project is a showcase of modern Android application development.

*   **UI:** Built entirely with **Jetpack Compose** for a declarative, reactive, and modern user interface.
*   **Architecture:** Follows a clean **MVVM (Model-View-ViewModel)** architecture with **Unidirectional Data Flow (UDF)** principles.
    *   The UI (View) observes state changes from the ViewModel.
    *   User actions are sent as events from the UI to the ViewModel.
*   **Dependency Injection:** Uses **Hilt** for managing dependencies, making the code more modular, testable, and robust.
*   **Asynchronous Operations:** Leverages **Kotlin Coroutines** and **StateFlow** to handle background tasks and manage state efficiently.
*   **Language:** Written entirely in **Kotlin**.
